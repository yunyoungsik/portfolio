Description

메인페이지 슬라이드는 Javascript를 사용하여 슬라이드 기능을 구현하고 GSAP와 Scroll Triger를 사용하여 각 섹션에 이미지와 어울리는 움직임을 표현하려고 했습니다. 상품페이지 핸드폰 정보가 들어있는 배열에서 각 핸드폰 정보를 꺼내어 처리하는데, 이를 foreach 루프로 수행합니다. 각 핸드폰 정보는 리스트 아이템(<li>)으로 만들어지며, 링크(<a>)와 이미지(<img>)를 포함합니다. 이미지를 가져오며, 제품 이름은 alt 속성에 들어갑니다. 나머지 핸드폰 정보인 제목, 설명, 가격 등은 <div class="text"> 안에 <p>와 <span> 태그를 사용하여 표시됩니다. 비교하기페이지 드롭다운 메뉴에서 변경 사항이 감지되면 이벤트를 트리거합니다. 이벤트는 선택된 옵션의 값을 기반으로 데이터를 필터링하고,해당 데이터에서 선택된 값을 기준으로 정보를 가져와 화면에 표시하는 역할을 합니다. 이벤트는 선택된 옵션의 값을 기반으로 데이터를 필터링하고, 해당 데이터에서 선택된 값을 기준으로 정보를 가져와 화면에 표시합니다. 로그인 사용자가 입력한 이메일과 비밀번호를 수집하여 이메일과 비밀번호로 데이터베이스에 저장된 회원 정보를 확인하기 위해 테이블에서 해당하는 데이터를 조회하는 SQL 쿼리를 생성합니다. 작성한 SQL 쿼리를 데이터베이스에 전달하고 실행합니다. 실행된 쿼리 결과를 확인하고 해당 결과를 기반으로 로그인 처리를 진행합니다. 그 밖에 ‘회원가입’, ‘게시판 글쓰기, 수정, 삭제’, ‘댓글’, ‘마이페이지 비밀번호 변경, 전화번호 변경, 생년월일 변경, 프로필사진 변경, 위시리스트’, ‘회원탈퇴’ 기능이 구현되어 있습니다.
